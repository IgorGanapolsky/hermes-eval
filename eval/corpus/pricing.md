# Acme Pricing & Limits

## Starter plan
- Up to 5 API keys per project.
- 10,000 requests per month included.
- Community support only.

## Pro plan
- Up to 50 API keys per project.
- 1,000,000 requests per month included.
- Email support with a 24-hour response time.

## Data residency
- All plans store data in the US (us-east-1).
- EU data residency is available on the Enterprise plan only.
